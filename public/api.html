Astrolux API

With our URL based customizability options, you can make Astrolux look completely different then it is!

Css Variables

--bg1

This is the primary background color for Astrolux.

Usage: {"--bg1":"#000000"}

--bg2

This is the secondary background color for Astrolux.

Usage: {"--bg2":"#101010"}

--color

This is the main text color Astrolux will use.

Usage: {"--color":"#ededed"}

--search-bg

This is the background the search box will use.

Usage: {"--search-bg":"#1a1a1a"}

--input-bg

This is the background color all other inputs will use.

Usage: {"--input-bg":"rgba(62, 62, 62, 0.13)"}

--placeholder

This is the color that placeholder text will use on input elements globally

Usage: {"--placeholder":"#8f8f8f"}

--background-image

This will allow you to change the gradient, bgimage, or to a solid color

Usage: {"--background-image":"linear-gradient(to bottom, #000000 60%, #101010 100%)"}

--font

This will allow you to change the font to any Google font or built in HTML font

Usage: {"--font":"'Poppins'"}



Url Querys

To customize Astroslate by URL, you can use the following URL querys.

Example: https://astroslate.one?query1=test&query2=test&query3=text
?css=

This allows the user to define and change the css variables mentoned above

Usage: ?css={"--bg1":"blue","--bg2":"white","--color":"green"}

?=search

Automaticlly opens the search iframe window with the defined url

Usage: ?search=https://silvereen.net

?titlecloak=

Changes the title of the page.

Usage: ?titlecloak=Astrolux

?imgcloak=

Changes the title image of the page to the defined url

Usage: ?imgcloak=1.png

?title=

Changes the "Astrolux" header to the defined text

Usage: ?title=Astrolux

?titlecloak=

Changes the title of the page.

Usage: ?titlecloak=Astrolux

?panickey=

Changes the panic key to the defined key

Usage: ?panickey=~

?panicsite=

Changes the site when the panic key is clicked to the defined url

Usage: ?panicsite=https://classroom.google.com
